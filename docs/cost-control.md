# Cost Control

MiniShop is intended to use AWS Free Tier resources where practical.

Planned cost-control practices include:

- Prefer Free Tier-eligible resources for learning environments.
- Use small, short-lived environments.
- Tag cloud resources for ownership and cost tracking.
- Configure billing budgets and alerts before provisioning.
- Stop or destroy resources when labs are complete.
- Avoid committing credentials or account information.

Exact resource limits and cleanup procedures will be documented with the Terraform and AWS phases.
